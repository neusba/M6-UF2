Enunciado 1

Crea una función que cree un array de 6 colores por nombre y que:

  • Diga si todos los colores son menores que marrón
  • Devuelva los colores que son menores que marrón
  • Devuelva los dos últimos colores independientemente de cuantos más haya antes
  • Agregue el color turquesa
  • Elimine el color verde

Enunciado 2

Crea una función Javascript que muestre por pantalla un ejemplo de cada una de las funciones de los arrays con el modelo siguiente: Array inicial -> copia del método y parámetros usados -> resultado

    Función at(): Returns an indexed element of an array

  • Función concat(): Joins arrays and returns an array with the joined arrays

  • Constructor: Returns the function that created the Array object's prototype

  • Función copyWithin(): Copies array elements within the array, to and from specified positions

  • Función entries(): Returns a key/value pair Array Iteration Object

  • Función every(): Checks if every element in an array pass a test

  • Función fill(): Fill the elements in an array with a static value

  • Función filter(): Creates a new array with every element in an array that pass a test

  • Función find(): Returns the value of the first element in an array that pass a test

  • Función findIndex(): Returns the index of the first element in an array that pass a test

  • Función flat(): Concatenates sub-array elements

  • Función flatMap(): Maps all array elements and creates a new flat array

  • Función forEach(): Calls a function for each array element

  • Función from(): Creates an array from an object.

  • Función includes(): Check if an array contains the specified element.

  • Función indexOf(): Search the array for an element and returns its position.

  • Función isArray(): Checks whether an object is an array.

  • Función join(): Joins all elements of an array into a string

  • Función keys(): Returns a Array Iteration Object, containing the keys of the original array

  • Función lenght(): Sets or returns the number of elements in an array

  • Función map(): Creates a new array with the result of calling a function for each array element

  • Función pop(): Removes the last element of an array, and returns that element

  • Función prototype: Allows you to add properties and methods to an Array object

  • Función push(): Adds new elements to the end of an array, and returns the new length

  • Función reduce(): Reduce the values of an array to a single value (going left-to-right)

  • Función reduceRight(): Reduce the values of an array to a single value (going right-to-left)

  • Función reverse(): Reverses the order of the elements in an array

  • Función splice(): Adds/Removes elements from an array

  • Función shift(): Removes the first element of an array, and returns that element

  • Función slice(): Selects a part of an array, and returns the new array

  • Función some(): Checks if any of the elements in an array pass a test

  • Función sort(): Sorts the elements of an array

  • Función toString(): Converts an array to a string, and returns the result

  • Función unshift(): Adds new elements to the beginning of an array, and returns the new length

  • Función valueOf(): Returns the primitive value of an array
